## Overview

Danny Chan Documentation provides a centralized space to create, organize, and publish your project documentation. You build structured docs with Markdown and MDX, collaborate with teams, and deploy changes seamlessly. This platform scales from small projects to enterprise knowledge bases.

## Getting Oriented with the Interface

Navigate the dashboard confidently using these steps. You access pages, edit content, and manage settings from a clean sidebar.

<Steps>
  <Step title="Access Dashboard" icon="layout">
    Log in at `https://docs.dannychan.com` and select your project.
  </Step>
  <Step title="Create Page" icon="plus">
    Click "New Page" in the sidebar. Choose a title and add frontmatter.
  </Step>
  <Step title="Edit & Preview" icon="edit">
    Use the split view to write Markdown and see live previews.
  </Step>
  <Step title="Publish Changes" icon="upload-cloud">
    Commit via Git or direct publish. View analytics post-deployment.
  </Step>
</Steps>

## Quick Integration Example

Embed your docs in apps using iframes or APIs. Here's how you fetch doc content.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://docs.dannychan.com/api/pages/introduction');
  const pageData = await response.json();
  console.log(pageData.content);
  ```
  ```python
  import requests
  response = requests.get('https://docs.dannychan.com/api/pages/introduction')
  page_data = response.json()
  print(page_data['content'])
  ```
</CodeGroup>
