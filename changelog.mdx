<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

- Added real-time collaboration for teams to edit docs simultaneously

- New search indexing for instant full-text results across all pages

## Improvements

- Updated UI components for better mobile responsiveness
- Improved version history with diff views

## Bug Fixes

- Fixed markdown rendering issues with nested lists
- Resolved export failures for large documentation sites

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core MDX documentation engine with 15+ components
- Full support for Steps, Tabs, Cards, and Update blocks

## Breaking Changes

- Removed legacy single-quote support in JSX components

## Initial Setup

- First stable release with complete parsing rules

## Upgrade Instructions

Follow these steps to update your Danny Chan Documentation instance.

<Steps>
  <Step title="Backup" icon="database">
    Create a full backup of your repository.

    ```bash
    git clone your-repo
    tar -czf backup.tar.gz docs/
    ```
  </Step>
  <Step title="Update Package" icon="package">
    Install the latest version using your preferred method.
  </Step>
  <Step title="Verify Changes" icon="check-circle">
    Run tests and preview the site.
  </Step>
</Steps>

<Tabs>
  <Tab title="npm" icon="package">
    ```bash
    npm install danny-chan-docs@latest
    npm run build
    npm start
    ```
  </Tab>
  <Tab title="yarn" icon="package">
    ```bash
    yarn add danny-chan-docs@latest
    yarn build
    yarn start
    ```
  </Tab>
  <Tab title="Docker" icon="docker">
    ```bash
    docker pull danny-chandocs:latest
    docker run -p 3000:3000 danny-chandocs:latest
    ```
  </Tab>
</Tabs>
